Keyboard Shortcuts
Set up custom keyboard shortcuts to quickly hide/show all tracked elements.
Available commands:
  • Hide all tracked elements
  • Show all tracked elements
How to set up hotkeys:
  1. Go to chrome://extensions/shortcuts
  2. Find "Element Toggle" in the list
  3. Click the pencil icon next to each command
  4. Assign your preferred key combinations
💡 Choose combinations that don't conflict with your other applications
